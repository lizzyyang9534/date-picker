## How to run this app
### npm
1. Run command `npm install`
1. Run command `npm run start`
1. Open the link http://localhost:3000 in your browser

### yarn
1. Run command `yarn`
1. Run command `yarn start`
1. Open the link http://localhost:3000 in your browser

## How to build this app
### npm
1. Run command `npm install`
1. Run command `npm run build`

### yarn
1. Run command `yarn`
1. Run command `yarn build`
